# 강화학습, 눈으로 보기 (Reinforcement Learning, Visualized)

밴딧부터 딥강화학습까지 — 에이전트가 **보상으로 스스로 배우는 과정**을 그리드월드·가치 히트맵·실시간 학습으로 직접 돌려보며 익히는 인터랙티브 학습 자료입니다. 정답을 알려주지 않고 보상만 주는 환경에서, 모든 Q-learning·가치반복·탐험 전략이 브라우저에서 실제 로직으로 동작합니다.

> 직접 만든 **비영리 학습용 자료**입니다. 특정 교재·강의와 무관하며, 설명·예시·코드는 모두 독자적으로 제작했습니다.

## 설계 원칙

- **실제 학습**: Q-learning·가치반복·탐험 전략이 모두 실제 코드로 동작 (하드코딩 결과 없음)
- **보상만으로**: 정답 경로를 주지 않고 보상·반복만으로 가치·정책이 드러나는 과정을 시각화
- **일관된 시각 언어**: 가치 히트맵(초록=좋음·빨강=나쁨), 정책 화살표(앰버), 에이전트 자취
- **단계별 분해**: 각 챕터 4스텝, 한 줄 직관 → 인터랙션 → 결론
